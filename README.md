# Intern Performance Analytics & Insights Dashboard

This project focuses on analyzing intern performance using a structured dataset.

## Sprint 1 Work
- Dataset creation and loading using Google Colab
- Data preprocessing (missing values, duplicates, data type validation)
- Summary statistics
- Preprocessing documentation

## Files
- Intern_Insight_dataset.csv → Dataset
- Internship_dataset.ipynb → Dataset creation (Google Colab)
- Data_cleaning.ipynb → Data preprocessing (Google Colab)
- Data Preprocessing Documentation.pdf → Documentation

## Tools Used
- Python
- Pandas
- NumPy
- Google Colab
